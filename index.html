¿Qué diferencia a JavaScript de cualquier otro lenguaje de programación?


JavaScript se diferencia fundamentalmente de otros lenguajes de programación por ser el único que tiene capacidad de ser compilado directamente en el navegador web. Esto permite que se pueda interactuar con la página web a tiempo real, haciendo que las páginas sean más dinámicas y responsivas.

Sus características más representativas son:

  • Lenguaje interpretado, no complilado: El código se lee y ejecuta línea por línea, en lugar de ser compilado en un programa ejecutable como en los lenguajes compilados. La ventaja de la interpretación es que puedes corregir errores y ajustar el comportamiento del código mientras se ejecuta.
  • Tipado dinámico: En JavaScript, las variables no tienen un tipo de dato fijo, pueden contener cualquier tipo de dato. Esto da mucha flexibilidad para trabajar con diferentes tipos de datos sin necesidad de definiciones rígidas. Sin embargo, es importante hay que ser cuidadoso para evitar confusiones y errores.
  • Funciones de primera clase: Las funciones en JavaScript no solo ejecutan tareas, sino que también pueden ser guardadas en variables, se pueden enviar a otras funciones como argumentos, e incluso crear nuevas funciones a partir de otras existentes. Esta flexibilidad permite un estilo de programación más funcional.

Además, muchas de las aplicaciones más populares del mundo, como Twitter, Gmail y Facebook, están creadas con código JavaScript. A pesar de que se pueda pensar que JavaScript solo sirve para animaciones web simples, estas empresas aprovechan potentes frameworks de Javascript, en esencia conjuntos de herramientas que ellas mismas crearon, para manejar tareas complejas y construir experiencias de usuario dinámicas. 

La versatilidad de JavaScript va más allá del desarrollo web. También es una excelente opción para aplicaciones móviles a través de frameworks como React Native, los desarrolladores pueden usar sus habilidades de JavaScript para crear aplicaciones para iOS y Android, ahorrando tiempo y recursos.

Incluso en tu flujo de trabajo diario, JavaScript puede ser una herramienta útil. Permite escribir pequeños scripts para automatizar tareas repetitivas, como hacer clic en un botón varias veces, agilizando tus tareas diarias.

¿Cuáles son algunos tipos de datos JS?


JavaScript admite varios tipos de datos para almacenar y representación información. Algunos de los más comunes son:

Primitivos:

  • Boolean: Valores TRUE o FALSE. Se puede usar para tomar decisiones en tu código.
  • Number: Números enteros (1, 2, 3) y decimales (3.14, 8.45). Se puede usar para realizar cálculos, como sumar, restar y multiplicar.
  • String: Cadenas de texto ("Hola, mundo"). Se puede usar para almacenar texto, como nombres, direcciones y mensajes.
  • Null: Un valor especial que indica la ausencia de un valor, valor nulo o vacío.
  • Undefined: Un valor especial que indica que una variable no ha sido inicializada.
  • BigInt: Números enteros grandes que no caben en el tipo Number. Se añade una 'n' al final del número o se usa la función BigInt().
  • Symbol: Valores únicos e inmutables que se utilizan como identificadores. 

No primitivos:

  • Object: Colecciones de datos que pueden contener propiedades y métodos. Puede servir, por ejemplo, para almacenar información de un usuario o una lista de productos.
  • Array: Un tipo especial de objeto que puede almacenar una lista ordenada de valores, accesible por índice. Por ejemplo, los nombres de los jugadores de un equipo de fútbol.
  • Function: Bloques de código que se pueden ejecutar para realizar una tarea específica. Por ejemplo, calcular el total de una compra.

El tipo de dato de una variable se puede determinar usando el operador typeof. Además, se puede convertir de un tipo de dato a otro usando diferentes métodos (se sugiere revisar información sobre el type casting). Recuerda que es importante elegir el tipo de dato adecuado para cada variable en tu código.

¿Cuáles son las tres funciones de String en JS?


Las funciones de string son métodos que se invocan sobre la variable que contiene la cadena. Estas funciones están integradas en JavaScript y permiten manipular y trabajar texto de diferentes maneras. Las tres más comúnmente utilizadas son la siguientes:

  • length: Es una propiedad, no una función, que devuelve el número de caracteres que contiene la cadena. También se puede utilizar con los array.

const str = "Hola, mundo";
const longitudCadena = str.lenght;
console.log(longitudCadena); // Imprime 10

  • indexOf(): Se utiliza para buscar la primera aparición de una subcadena (searchValue) dentro de la cadena original (str) y devuelve su posición (índice) dentro de la cadena. Si la subcadena no se encuentra, la función devuelve -1.

const str = "Hola, mundo";
const indice = str.indexOf("mundo");
console.log(indice); // Imprime 5

  • toUpperCase() y toLowerCase(): Transforman todo el contenido de la cadena a mayúsculas o minúsculas, respectivamente. 

const str = "Hola, mundo";
const strMayusculas = str.UpperCase();
console.log(strMayusculas); // Imprime "HOLA, MUNDO"

Además de las anteriores, es importante conocer que existen muchas otras funciones sumamente útiles para tratar los datos de tipo string. A continuación se mencionan varias:

  • charAT(): Obtiene el caracter en la posición indicada por el índice.
  • trim(): Elimina los espacios en blanco al principio y final de la cadena.
  • includes(): Comprueba si la cadena contiene una subcadena específica. Devuelve el valor verdadero o falso.
  • replace(): Reemplaza una subcadena por otra dentro de la cadena original.
  • slice(): Extrae una subcadena de la cadena original especificando el índice de inicio y fin.
  • split(): Divide la cadena en un array de subcadenas separadas por un caracter o una cadena específica.
  • startsWith() y endsWith(): Comprueba si la cadena comienza o acaba con una subcadena específica. Devuelve valor verdadero o falso.
  • repeat(): Repite la cadena un número específico de veces.
  • concat(): Concatena dos o más cadenas.

¿Qué es un condicional?


Las condicionales son una estructura de control que nos permiten tomar decisiones y controlar el flujo de un programa dependiendo de si una condición se cumple o no.

Tipos de condicionales:

  • if: Se utiliza para evaluar una única condición.
  • else if: Se utiliza para evaluar otra condición si la primera no se cumple.
  • else: Se utiliza como último recurso, si ninguna de las condiciones anteriores se cumple.

Para comparar valores, se utilizan los siguientes operadores:

  • === : Igual en valor y tipo de dato.
  • !== : Diferente en valor o tipo de dato.
  • >= : Igual o mayor que.
  • <= : Igual o menor que.

Para crear reglas más complejas, se pueden combinar condicionales y utilizar operadores como and (&&) o or (||).

Ejemplo

Imagina que quieres saber si una persona puede votar. Para verificar si es igual o mayor de edad, puedes usar una condicional, y así saber si tiene derecho al voto.

const edad = 17;
if (edad >= 18) {
    console.log("Puedes votar");
} else {
    console.log("Lo siento, aún no puedes votar");

Se define una variable edad y se le asigna el valor de 17. Se inicia una declaración condicional con la palabra clave if seguida de la condición entre paréntesis. Esta condición verifica si la edad es mayor o igual a 18.
Si la condición es verdadera, se ejecuta el bloque de código dentro del primer par de llaves, diciendo que la persona puede votar. 
Si la condición en el if es falsa, se ejecuta el bloque dentro del else, indicando que aún no puede votar.

Ejercicio recomendado

Usa condicionales para mostrar un mensaje diferente dependiendo de la calificación:

  • Si la calificación es mayor o igual a 90, mostrar "Excelente!".
  • Si la calificación está entre 70 y 89, mostrar "Buen trabajo!".
  • Si la calificación está entre 50 y 69, mostrar "Aprobado.".
  • Si la calificación es menor a 50, mostrar "Desafortunadamente, no has aprobado.".

¿Qué es un operador ternario?


El operador ternario, o condicional ternario, es una herramienta poderosa que permite tomar decisiones dentro de una sola línea de código. Se utiliza para evaluar una condición y devolver uno de dos valores dependiendo del resultado.

Su sintaxis es la siguiente:
condicion ? valorSiVerdadero : valorSiFalso

Siendo la condición, la expresión que se evalúa, el valorSiVerdadero el que se devuelve si la condición es true y valorSiFalso, si la condición es false.

Las principales ventajas del operador ternario son que permite escribir código más compacto y legible al estar expresado en una única línea. Supone una alternativa sencilla a las estructuras if / else cuando la decisiones son simples, siendo útil para asignar valores a variables dependiendo de una condición. 
Se recomienda no hacer condicionales compuestas utilizando el operador ternario, pero en caso de hacerla, resulta útil encapsular con paréntesis la condiciones.

Ejemplo

const edad = 18;
const mensaje = edad >= 18 ? "Puedes votar" : "Lo siento, no puedes votar";

Este ejemplo es el mismo que el explicado en el apartado anterior sobre las condicionales. Su estructura es mucho más sencilla al utilizar el operador ternario.

Ejercicio recomendado

Dada la siguiente condicional, conviértela en un operador ternario:

let numero = 10;
let resultado;
if (numero > 0) {
    resultado = "Positivo";
} else {
    resultado = "Negativo";
}

¿Cuál es la diferencia entre una declaración de función y una expresión de función?


En JavaScript hay dos maneras comunes de definir funciones: mediante declaraciones de función y mediante expresiones de función. Las diferencias entre ambas recaen en cómo son tratadas por el intérprete de JavaScript y cómo se comportan en el código.

Declaración de función:

  • Se utiliza la palabra clave function.
  • Se define un nombre para la función.
  • Se puede usar antes de ser definida.
  • Se asigna a un objeto o a la variable global.

Ejemplo

function saludar () {
    console.log("¡Hola!");
}
saludar();


Expresión de función:

  • Se define como una expresión dentro de otra expresión.
  • No es necesario un nombre.
  • No se puede usar antes de ser definida.
  • Se puede asignar a una variable o utilizar como valor inmediato.

Ejemplo

const saludar = function () {
    console.log("¡Hola!");
}
saludar();


Las declaraciones de función se utilizan para definir funciones que se van a utilizar en diferentes partes del código, mientras que las expresiones de función se utilizan para definir funciones que se van a utilizar de forma puntual o dentro de otra expresión.

Se recomiend usar declaraciones de función para las funciones principales del programa y, expresiones de función, para pequeñas funciones o que solo se vayan a utilizar una vez.

¿Qué es la palabra clave "this" en JS?


La palabra clave this en JavaScript se utiliza para referirse al objeto actual en el contexto de una función. El valor de this cambia dependiendo de cómo y dónde se llame a la función.

En métodos de objetos, this hace referencia al objeto en el que se llama al método.

Ejemplo

const persona = {
    nombre: "Mikel",
    saludar() {
        console.log("Hola, me llamo ${this.nombre}");
    },
};


En funciones normales hace referencia al objeto global.

Ejemplo

function saludar(){
    console.log("Hola, soy ${this.nombre}");
}


En funciones flecha this no cambia y hereda el valor de la función padre.

Ejemplo

const persona = { 
    nombre: "Mikel",
    saludar() {
        const flecha = () => console.log("Hola, mi nombre es ${this.nombre}");
        flecha();
     },
};
persona.saludar(); // Imprime "Hola, mi nombre es Mikel"
